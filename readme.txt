COP3504c - Lab 10: Meme Generator
Overview: In this lab, students will use an external library to create a meme generator library and executable.
          The purposes of this assignment is to give students practice with setting up, importing, using, and writing
          libraries in C++.
Executable: Should function with just file and top text, but should also accept a partial/full complement
          ./memeify doge.jpg "Such memes"
          ./memeify doge.jpg "Such memes" "wow" 360 90 120 360
The new image will be saved as "doge-meme.jpg"
